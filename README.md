# Discussion

Course discussion will be held in the [issues](https://github.com/STAT540-UBC/Discussion/issues) of this repository. To start a new thread, create a new issue. Tag your peers using their handle—@gcohenfr, for example—to get their attention.

Lots of the good stuff will end up in [closed issues](https://github.com/STAT540-UBC/Discussion/issues?q=is%3Aissue+is%3Aclosed), so dont' forget to look there! Or, more generally, use the "Search" feature on this repository to find things.
